Problema do cliente/servidor implementado com socket com duas opçãos:

- TCP;
- UDP;

1- servidor é iniciado com uma das opções UDP ou TCP;
2- Cliente é iniciado UDP ou TCP;
3- O cliente envia mensagens sucessivas ao servidor, que mostra o recebimento de cada uma;
4- o programa permanece em loop;
