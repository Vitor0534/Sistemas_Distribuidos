# Português:
implementação de comunicação cliente/Servidor simples. O servidor é executado primeiro e permanece a espera da conexão do cliente. quando o cliente é executado ele se conecta ao servidor e envia uma mensagem de confirmação de conexão ao servidor.

# English:
implementation of simple client/server communication. The server runs first and remains to wait for the client to connect. when the client runs it connects to the server and sends a connection confirmation message to the server.
