O programa consiste de uma Hash para quebra de senhas. o programa faz uso de uma wordlist para selecionar uma senha e chutar a mesma posteiormente. Isso é feito atravez de um web_server, o cliente acessa o servidor e solicita o serviço de quebra senha. Caso a senha seja quebrada é mostrada para o cliente que a mesma foi quebrada.
