Autor: Vitor de Almeida Silva

Este programa implementa um webService SOAP para um e-comerce. o programa é compostos pelos seguintes pontos:

1- Itens: são os intens que estão a venda no no site do servidor. o servidor tem total liberdade para adição de novos itens, exclusão, alteração, listagem de itens, visualizar um item específico e listar compras realizadas por clientes;

2- Servidor: Tem controle do estoque de itens e pode alterar as informações dos mesmos. Responsável por forncer o serviço de venda de produtos;

3- Cliente: O cliente tem permição para listar produtos, visualizar um produto específico e comprar um produto.


o programa funciona da seguinte forma:

1- o usuário deve escolher entre servidor e cliente;

2- se o usuário escolheu servidor, ele pode realizar todas as ações do servidor no e-comerce, inserindo ou retirando produtos e alterando preços;

3- se o usuário escolheu cliente, ele pode comprar, listar e visualizar produtos inseridos anteriormente pelo servidor;

4- o programa permanece em loop.
